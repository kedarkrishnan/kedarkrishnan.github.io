    Creating revealing header and footer with CSS

    The effect we are looking for is like slide in and out of the header and footer. So when the main page is scrolled up the top header of the page slides under the main page while the footer is releaved as though it was under the main page.

    See the Pen Revealing header and footer effect with simple CSS by Kedar Krishnan (@kedarkrishnan) on CodePen.

    This can be obtained using the z-index property of css and fixing the layers we want to reveal. We create two layers one with z-index of 1000 which would act as the top layer (section tag, .content class) and another with the z-index of 999 (header and footer tags, .header and .footer class) which would go below the top layer. The position of the bottom layer is set to fixed so that they remain at the top and bottom of the page respectively. The header and the footer are hidden by the section layer because of this setup, yeap we need to provide a background color for the top layer.

    As the trick is to cover the bottom layer it is important that the height of the section is bigger than the height of the header and footer. Now to display the header and footer we move the content of the top layer by giving it top and bottom margin and we are all set. As the page is scrolled the top layer margin section moves up and the content section covers the header. In the same way as we reach the end of the page due to the bottom margin the footer section is revealed.
