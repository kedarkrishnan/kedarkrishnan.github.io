Introduction

As I moved towards the front-end technologies I came across Sublime text editor and have really fallen in love with the editor. It's simple yet powerful tool for working on a number of languages. It has a number of packages which can be added as needed enabling more and more features for the tool. Though installing packages is a bit tricky and knowing that there are such powerful packages is another thing. The intent of this Bare bones is to get new developers started with the editor.
